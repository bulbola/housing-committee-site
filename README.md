# Информационный портал жилищного комитета

Учебный сайт для эксплуатационной практики.

## Что реализовано

- 7 основных страниц сайта.
- Карта сайта.
- Страница 404.
- Стандартный стиль.
- Версия для слабовидящих.
- Новости.
- Услуги.
- Документы.
- Форма обращения граждан.
- Поиск по сайту.
- Личный кабинет.
- 3 роли пользователей: гражданин, сотрудник, администратор.

## Запуск через XAMPP

1. Установить XAMPP.
2. Запустить Apache.
3. Скопировать папку `housing_committee_site` в `C:\xampp\htdocs\`.
4. Открыть в браузере:
   http://localhost/housing_committee_site/
